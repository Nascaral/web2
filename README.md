## Overview

Bridge is a financial clarity platform that allows users to connect all their financial accounts, track spending, monitor net worth, and make better money decisions. The platform prioritizes security, privacy, and user experience.

## Features

- **Account Aggregation**: Connect multiple bank accounts, credit cards, and investment accounts
- **Real-time Syncing**: Transactions sync in real-time, not nightly
- **Wealth Tracking**: Monitor net worth and financial health
- **Spending Insights**: Automatic transaction categorization and spending analysis
- **100% Free**: No hidden fees or premium tiers
- **Bank-level Security**: Industry-standard encryption and security protocols
- **Privacy-first**: User data is never sold or shared

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, but recommended for development)

## Browser Support

- Chrome/Edge (last 2 versions)
- Firefox (last 2 versions)
- Safari (last 2 versions)
- iOS Safari (last 2 versions)
- Android Chrome (last 2 versions)

## Security

- No sensitive data is stored in localStorage
- All forms use HTTPS in production
- Input sanitization on all user inputs
- Content Security Policy headers recommended

## Responsive Design

The site uses a mobile-first approach with breakpoints:

- **Mobile**: Less than 768px
- **Tablet**: 768px - 1024px
- **Desktop**: Greater than 1024px

## Testing

### Manual Testing Checklist

- All links work correctly
- Forms validate properly
- Responsive design works on all breakpoints
- Keyboard navigation works
- Screen reader announces content correctly
- Images load and have alt text
- No console errors
